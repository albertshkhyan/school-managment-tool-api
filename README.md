# School Management Tool (API)

Welcome to the School Management Tool (API) project! This application provides a backend API for managing students, teachers, classes, and grades within a school.

## Features

- Create, read, update, and delete operations for students, teachers, classes, and grades.
- GraphQL API for efficient data querying and manipulation.
- Input and output DTOs for structured data exchange.
- Scalable architecture using NestJS.

## Technologies Used

- **Framework**: NestJS
- **GraphQL**: Apollo Server, GraphQL
- **Database**: PostgreSQL (or any supported by TypeORM)
- **Validation**: class-validator
- **Utilities**: rxjs, uuid
- **Testing**: Jest, Supertest
- **Linting & Formatting**: ESLint, Prettier
- **Version Control**: Git, GitHub

## Getting Started

To get started with the School Management Tool (API) project, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/albertshkhyan/school-managment-tool-api
   ```


2. Navigate to the project directory:

   ```bash
   cd school-managment-tool-web
   ```

3. npm Install dependencies:

   ```bash
   npm install
   ```
